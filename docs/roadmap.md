# Roadmap

Localdeck 1.0 focuses on zero-config discovery of currently running local web apps.

## After 1.0

- Improve framework detection with safe response headers and known local-only endpoints.
- Display confidence and evidence details without cluttering cards.
- Add Playwright E2E tests for loading, filtering, rescan, card clicks, and endpoint chips.
- Add a short-lived favicon cache with eviction.
- Improve Docker/Compose grouping by project and service.
- Add CI release automation and signed release images.
- Add runtime hints for Ruby, PHP, Java, Go, Rust, and .NET local servers.

## Still Out Of Scope

- Container management actions.
- Persistent dashboards or bookmarks.
- User accounts or authentication.
- Uptime history, alerts, and monitoring.
- Remote host discovery.
